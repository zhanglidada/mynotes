**工作上：**
年初的时候想了几个新项目，包括《sql 函数内联》，《对象回收增强》
年中的时候想了两个，分别是《bifun 优化》以及《sql 函数批量》

其中 sql 函数内联写了两个专利，不过因为优先度不高所以暂时搁置了
对象回收增强感觉不太好实现，也暂时搁置

bifun 优化主要是使用 simd 指令集对 bifun 函数的实现进行改造，这个项目在实现的过程中难点挺多，目前也仅仅是优化了几个小点，更多还是探索性质。后续可以继续优化其他 bifun 函数，同时也可以进一步进行向量化执行改造。

sql 函数批量，这个也是自己根据 sql 函数内联项目进行发散思考想到的改造。项目刚开始的时候感觉难度挺高，但是在逐渐改造的过程中也增加了对 sql 函数整体执行流程的熟悉，后续也就感觉没那么难了。（说明任何看起来很难的事情都要勇敢去尝试，随着不断的思考、讨论，对相关方向也就会从陌生到熟悉）

25 年的最后还做了一个指令融合的尝试，测试后性能提升也挺明显的，这个在后续也可以进一步扩展改造。
